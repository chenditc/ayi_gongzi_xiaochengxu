# 日历工资计算（微信小程序）

本项目是一个**无后端**的微信小程序：用户只需在日历上**点选她实际上班的日期**，系统会自动把当月的**法定节假日（300%工资）**计入带薪，并按规则计算预估工资。

## 计算规则（法定节假日）

- **法定节假日不上班**：按 **1 倍日薪**计（视为带薪 1 天）
- **法定节假日上班**：按 **3 倍日薪**计（= 1 倍带薪 + 额外 2 倍加班增量）

用“天数单位”表达：

- **工作日上工天数**：当月**非**法定节假日且用户点选为上班的天数
- **带薪休假天数**：当月法定节假日总天数（自动，不需点选）
- **节假日加班天数**：当月法定节假日里用户点选为上班的天数（自动识别）

公式：

> 计薪单位天数 = 工作日上工天数 + 带薪休假天数 + 2 * 节假日加班天数  
> 日薪 = 月薪 / 应出勤天数  
> 预估工资 = 计薪单位天数 * 日薪

参考页面：`calendar_salary_calculator.html`（仅作为功能/UI参考，小程序运行不使用该文件）。

## 打开方式（微信开发者工具）

1. 打开微信开发者工具
2. 选择“导入项目”
3. 项目目录选择本仓库根目录
4. AppID 选择“测试号”或保持默认（本项目已配置 `touristappid`）
5. 运行即可

## 功能清单（对照测试）

- **月份切换**：上月/下月（跨年自动处理）
- **单日选择**：点击日期格子切换选中/取消
- **按星期列选择**：点击周标题（如“周一”）一键选择/取消本月该星期列的所有日期
- **本月全选 / 取消本月**：快速批量操作
- **清空数据**：清除本地保存的选择日期、月薪、应出勤天数
- **输入**：月薪、应出勤天数（0-31）
- **计算**：实时显示“工作日上工/法定节假日带薪/节假日加班(额外×2)”拆分，以及 `预估工资 = 计薪单位 / 应出勤天数 * 月薪`
- **分享工资单**：分享“本月选择 + 月薪/应出勤”组合；他人通过分享卡片打开后可看到相同月份与结果，并自动保存到本机
- **本地持久化**：使用 `wx.setStorageSync` 保存；启动自动恢复；带基础版本号与日期合法性校验
- **明细**：可展开/收起已选日期列表（含星期）
- **法定节假日标识**：日历日期右上角显示“法”；点选该日期会按“节假日加班（额外×2）”计入

## 分享工资单（无后端）

在“计算结果”卡片中点击 **分享工资单**，小程序会生成一个包含当前月份数据的分享链接，主要包含：

- **月份**：年/月
- **本月已选日期**：用日期号（1-31）编码
- **输入项**：月薪、应出勤天数

他人打开分享卡片后，会看到同样的月份与计算结果，并会**自动保存到本机**（无额外提示）。

注意：由于本项目无后端，分享链接里会包含月薪/应出勤等输入内容，请只分享给可信任对象。

## 文件结构

- `app.json` / `app.js` / `app.wxss`：小程序全局配置与样式
- `pages/index/`：主页面（UI + 逻辑）
- `utils/date.js`：日期工具函数
- `utils/storage.js`：本地存储封装（版本号 + 校验）
- `utils/holidays.js`：法定节假日（300%工资）数据（目前仅 2026）与判断工具
- `utils/share.js`：分享链接的编码/解析工具（无后端）
