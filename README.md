# 日历工资计算（微信小程序）

本项目是一个**无后端**的微信小程序：用日历勾选当月出勤日期，并按公式计算预估工资：

> 预估工资 = 已选天数 / 应出勤天数 * 月薪

参考页面：`calendar_salary_calculator.html`（仅作为功能/UI参考，小程序运行不使用该文件）。

## 打开方式（微信开发者工具）

1. 打开微信开发者工具
2. 选择“导入项目”
3. 项目目录选择本仓库根目录
4. AppID 选择“测试号”或保持默认（本项目已配置 `touristappid`）
5. 运行即可

## 功能清单（对照测试）

- **月份切换**：上月/下月（跨年自动处理）
- **单日选择**：点击日期格子切换选中/取消
- **按星期列选择**：点击周标题（如“周一”）一键选择/取消本月该星期列的所有日期
- **本月全选 / 取消本月**：快速批量操作
- **清空数据**：清除本地保存的选择日期、月薪、应出勤天数
- **输入**：月薪、应出勤天数（0-31）
- **计算**：实时显示 `已选天数 / 应出勤天数 * 月薪 = 结果`
- **本地持久化**：使用 `wx.setStorageSync` 保存；启动自动恢复；带基础版本号与日期合法性校验
- **明细**：可展开/收起已选日期列表（含星期）

## 文件结构

- `app.json` / `app.js` / `app.wxss`：小程序全局配置与样式
- `pages/index/`：主页面（UI + 逻辑）
- `utils/date.js`：日期工具函数
- `utils/storage.js`：本地存储封装（版本号 + 校验）
